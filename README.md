INSTRUCTIONS TO RUN THE PROJECT:

1. Ensure you have Node.js
2. Download the Zip file off of github
3. Unzip the folder
4. Open it in an IDE
5. Type the following:

Type “npm install pg”
Type "npm install pug"
Type "npm install express"
Type "npm express-session"
Type “npm install”
Type “npm start”
